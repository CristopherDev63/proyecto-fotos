# Cómo Contribuir

¡Gracias por tu interés en contribuir a este proyecto!

## Reportando Errores

Si encuentras un error, por favor, abre un "issue" en GitHub y describe el problema. Asegúrate de incluir los pasos para reproducirlo.

## Sugiriendo Mejoras

Si tienes una idea para una mejora, abre un "issue" para discutirla. 

## Pull Requests

1.  Haz un "fork" del repositorio.
2.  Crea una nueva rama para tus cambios.
3.  Haz tus cambios y "commit".
4.  Haz un "push" a tu "fork".
5.  Abre un "pull request".
